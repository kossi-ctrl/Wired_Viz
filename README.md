# Wired_Viz
Create a website to analyze data from Magazine Wired USA. 

# Objectifs 

Ce Projet a pour objectifs, l’extraction automatisée de données depuis des sites web ou plateformes,
l’utilisation des bases de données relationnelles et documentaires, 
la visualisation de données statistiques et enfin l’analyse d’image en pratique 
En théorie, il s‘agira de poser un regard critique sur la culture, 
la visualisation de l’information et les recherches à l’ère des IA.
